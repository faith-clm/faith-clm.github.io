10 COMPUTER TERMS

Byte
small unit of data storage; 8 bits; usually holds one character

Boot
the process of loading or initializing an operating system on a computer; usually occurs as soon as a computer is turned on

GUI
Graphical User Interface, a computer program that enables a person to communicate with a computer through the use of symbols, visual metaphors, and pointing devices.

Database
a large structured set of data; a file that contains numerous records that contain numerous fields

Modem
a hardware device that allows computers or routers to receive and send information over a cable or telephone line.

Cache
a collection of items of the same type stored in a hidden or inaccessible place.

Port
an interface or connection point between a computer and an internal or external device.

Malware
any intrusive software developed by cybercriminals (often called hackers) to steal data and damage or destroy computers and computer systems.

Firewall
a network security device that prevents unauthorized access to an operating system.

Motherboard
the main circuit board that connects the various parts of a computer.
